prototype
+   case study

+   ui/ux

SnapSpot is a location scouting and social networking mobile application geared towards working professional photographers. It provides users with the ability to easily scout locations with the integration of GPS and location services by looking up locations based on keywords or location names. So how might we be able to empower professional photographers to minimize the time they spend scouting locations through an application that allows them to easily find and share locations in a simplified manner?

+   collaborators _ ashley bass & carol newhard
+   methodologies _ interviews, literature review, personas, prototyping, usability testing

To create a prototype that would solve our how might we statement, we had to begin with looking at the current market for any similar applications and completing user research. To find out the similarities and differences between the current applications on the market, we created a list to compare and contrast the three that we discovered.
